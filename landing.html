Deploy Anywhere. In Seconds.

No cloud dependencies. No complex setup. Choose the method that fits your operation.

⚡

Single EXE File

Recommended

Double-click to run. No installation. No dependencies. Windows, Mac, and Linux builds.

npm run build:all
📄

Static HTML File

One HTML file that opens in any browser. Email it, put it on a USB drive, host it anywhere.

npm run build:singlehtml
🐳

Docker Container

Containerized deployment with docker-compose. Persistent volume for database.

docker-compose up -d
🥧

Raspberry Pi

Vehicle-mounted edge deployment. Runs on less than 50MB RAM. Systemd service included.

<50MB RAM idle
☁️

Cloud Deploy

One-click deploy to DigitalOcean, Render, Fly.io, or Vercel. Full server deployment in under 2 minutes.

🐧

Linux Systemd

Permanent server installation with systemd service. Auto-start on boot. Journal logging.
